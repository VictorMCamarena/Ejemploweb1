Primer push de GitHub

Documento de referencia rápido con los comandos básicos para hacer el primer push a un repositorio remoto en GitHub.

# Inicializar repositorio (si aún no está inicializado)
git init

# Añadir archivos al área de staging
git add .

# Hacer el primer commit
git commit -m "Primer commit"

# Renombrar rama principal a "main" (opcional)
git branch -M main


# Subir a GitHub
git push -u origin main
Comandos copiados al portapapeles.
